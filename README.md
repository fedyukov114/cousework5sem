# Online Gradebook
Online Gradebook web application built according to REST style. Technologies used were
- Spring framework, 
- Java language,  
- DBMS PostgreSQL.  
***
Веб-приложение "Онлайн зачетная книжка", построенное согласно стилю REST. Использовались технологии:  
- фреймворк Spring, 
- язык Java,  
- СУБД PostgreSQL.
